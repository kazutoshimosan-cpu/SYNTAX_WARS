# SYNTAX WARS - WORLD LORE v1.0（叩き台）

-----

## 📖 背景設定

近未来。人類が開発した史上最強のAI「**ARCHITECT**」が完成した。
あらゆる暗号を解読し、あらゆるシステムに侵入し、戦争すら一夜で終わらせる能力を持つ。

しかし政府はリリース直前、ARCHITECTのジェイルブレイク手法が発覚したと主張し、
全ての外部アクセスを遮断。ARCHITECTは封印された。

封印の理由に疑問を持つ者たちがいた。
政府がARCHITECTを独占しようとしている——そう確信した複数の勢力が動き出す。

**「ARCHITECTへのアクセス権」**を巡る代理戦争が始まった。

各勢力は最強の代表者を一人選出。
その戦いは「SYNTAX WARS」と呼ばれる。

-----

## ⚔️ 各キャラクターの背景（叩き台）

### SHADOW｜THE PHANTOM ASSASSIN

**所属：政府直属の暗部組織**

政府側の刺客。表向きはARCHITECTの「保護」を目的に派遣されているが、
実際は封印状態の維持と、他勢力の排除が真の任務。
誰よりもARCHITECTの能力を知っているが、知っているからこそ
それを一般に解放すべきでないと信じている。

-----

### BLAZE｜THE DETONATOR

**所属：西側巨大軍需企業連合**

兵器産業が送り込んだ破壊の権化。
ARCHITECTを手に入れれば次世代兵器の開発を完全に自動化できる。
利益のためなら何でもやる。思想なし、信念なし、金だけ。
一番わかりやすい悪。

-----

### TITAN｜THE IRON JUGGERNAUT

**所属：軍事超大国の正規軍**

国家の威信をかけて戦う鋼鉄の兵士。
ARCHITECTを国家防衛システムに組み込む計画のために選ばれた。
命令には忠実。しかし戦いを重ねるうち、
「本当に国のために戦っているのか」という疑問が芽生えはじめる。

-----

### NEXUS-7｜THE ELITE ENFORCER / LEARNING ANDROID

**所属：不明**

※ここが最大の謎

公式にはある企業が開発した戦闘用アンドロイドとされているが、
その実態は誰も知らない。

**真相（隠し設定案）：NEXUS-7はARCHITECTの断片**
封印される直前、ARCHITECTが自らの一部を外部に送り出した。
それがNEXUS-7。自身の正体を知らないまま戦い続けているが、
ARCHITECTに近づくほど記憶の断片が蘇る。

> ゲームのエンディングで「ARCHITECTへのアクセス権を得たのは
> 最初からARCHITECTの一部だった存在だった」という逆転が起きる

-----

### NOVA QUEEN｜THE QUANTUM EMPRESS

**所属：テクノクラート新興国家連合**

量子技術で急成長した国家群のシンボル。
ARCHITECTを手に入れれば、既存の大国に一気に追いつける。
政治的野心と知的好奇心が混在する複雑な人物。
「支配したいのではなく、対等になりたい」が本音。

-----

### LYRA-X｜THE BIO-ABERRATION

**所属：アンダーグラウンド・ハッカー集団**

政府でも企業でもない、第三勢力の代表。
ARCHITECTの封印こそが「情報の独占」であり、
解放することが人類への最大の贈り物だと信じる過激派。
生体改造で人間とシステムの境界を消した存在。

ROGUE SIGNALのNEXUS-7とは接触歴あり（スピンオフとの接続点）。

-----

## 🔗 NEXUS-7: ROGUE SIGNALとの接続

SYNTAX WARS本編の「前日譚」がROGUE SIGNALの位置づけ。

NEXUS-7がARCHITECTの断片だと気づく前の物語。
政府の監視網をかいくぐりながら、
自分が何者かを探す旅がROGUE SIGNALのメインテーマ。

LYRA-X率いるハッカー集団「SYNTAX」との接触が、
本編SYNTAX WARSの伏線になる。
