# SYNTAX WARS — エンディング演出＆分岐EN UI 設計

> キャラエンディング（コミックパネル）の再生フローと、NEXUSの分岐EN（解放／封印）の
> 選択UI＋全6体のエンディング・ギャラリー。**ハイブリッド方式**＝初回は重い一度きりの
> 選択、以後はギャラリーで両方鑑賞。全Godot表示層・**JS脳（combat）不変**。
>
> 関連＝パネル内容 `docs/ending_panels.md` ／ 弧 `docs/character_arcs.md` ／ ロア `SYNTAX_WARS_LORE_v2.1.md`。
> 作成 2026-06-19。ステータス：**設計＝確定**／実装は作画完了後（UIはプレースホルダで先行可）。

## 1. 概要

アーケード（5連戦）制覇で流すキャラ別エンディング。NEXUSだけ「解放するか否か」の
**分岐EN**を持つ。見せ方は**ハイブリッド**：初回制覇は心理戦らしい"重い一度きりの選択"、
以後はギャラリーで両分岐・全6体EDを鑑賞できる。

## 2. 体験のねらい（Player fantasy）

- NEXUS初制覇の「解放／封印」＝**選択の重み**を一度だけ本気で背負わせる（後から取り消せない初回）。
- 以後は収集欲（両EN／全6体ED）で回遊。**全キャラ制覇チャレンジ**とも相互に噛む。

## 3. フロー

1. アーケード制覇 → エンディング再生（共通パネル：`ending_panels.md`）。
2. **NEXUSのみ**：クライマックス（P5「解放するか否か——お前が選べ」）直後に**選択画面**。
   - 初回＝二択（解放／封印継続）。**制限時間なし**・コミット式（§5）。
   - 選択 → 対応する最終パネル（P6 解放／P7 封印）。
3. 選んだENを `Progress` に記録（seen）＋**初回選択を"あなたのcanon"として保持**。
4. 2回目以降のNEXUS制覇＝**あなたのcanon ENを再生**（選び直しは強制しない）。
5. もう片方のEN＝**ギャラリーで解放・鑑賞**（初回の重みを侵さない）。
6. 他5体＝制覇で各1EDを解放（分岐なし＝選択画面スキップ）。

## 4. 選択画面の見せ方

- 端末の前、暗転ベース。ARCHITECTの台詞が残る中、**二択だけ**を提示（HUD／タイマー無し）。
- **解放＝広がる光（シアン拡散）／封印＝閉じて収束**、の左右対比モチーフ。
- **重みの装置＝コミット式**（推奨：長押しで確定、または 選択→"本当に？"の一拍）。
  即タップ事故を防ぎ、"選んだ"実感を作る。

## 5. ギャラリー（ARCHIVE／回想）

- タイトル または リザルト後メニューから入る。
- 6体ぶんのEDを一覧。**鑑賞済のみ再生可・未解放は「？？？」**。
- NEXUSは**2分岐スロット**（初回で片方、ギャラリーでもう片方を解放）。
- 全EDコンプ＝収集フック（全キャラ制覇チャレンジと相互補完）。

## 6. セーブデータ（Progress autoload／`user://progress.cfg`）

- `ending_seen.<char>`：各キャラED鑑賞フラグ。
- `nexus_ending_liberate` / `nexus_ending_seal`：NEXUS両分岐の鑑賞フラグ。
- `nexus_canon`：初回に選んだ分岐（"あなたのcanon"・上書きしない）。
- 既存Progress方針に合わせ **NORMAL以上のみ記録**（しばりチャレンジ実績と同じ器）。

## 7. エッジケース

- NEXUS初回選択を中断（アプリ落ち）＝未選択扱い → 次回再提示。
- canon確定後にギャラリーで反対ENを見ても、`nexus_canon` は上書きしない。
- 作画未完＝プレースホルダパネルで成立（UI／フローは先に組める）。
- 他5体は分岐なし＝選択画面を出さない。
- EASYなど記録対象外の難易度＝EDは流すが解放フラグは保存しない（要・最終確認）。

## 8. 依存

- 既存 `Progress` autoload（実績セーブ）／アーケード進行（`battle_controller`/`roster_data`）
  ／`Transition`・`UiTheme` autoload（遷移・テーマ）／`ending_panels.md`（パネル内容）。
- **JS脳（combat）とは無関係＝再ベイク不要**。

## 9. チューニングノブ

- コミット方式（長押し秒数／確認一拍）。
- 選択画面・各パネルの演出尺。
- ギャラリー導線位置（タイトル or リザルト後 or 両方）。

## 10. 受け入れ基準

- NEXUS制覇 → 選択が**一度だけ**出る → 選んだ分岐の最終パネルが流れる → フラグ保存。
- 2回目制覇＝**canon EN再生**（選択を再強制しない）。
- ギャラリー：未解放＝「？？？」／解放済＝再生可／NEXUSは両分岐スロット。
- 他5体＝制覇で各ED解放・分岐なし。
- 全部Godotで完結・**JS脳不変**・パースclean（exit0）。
