## 6. ゲームプレイ録画（取得方法）

実機素材（#1アリーナ／#4 SELECT／#5 同時開示／KO／CROWD ON FIRE）は **手収録でなく自動録画で取得**（制御・画質・撮り直しで優位）。

- **手段**＝Godot の Movie Maker（`--write-movie`）＋ 自動操作ハーネス（`_`始まりの使い捨て＝gitignore対象）。`battle_controller._on_player_move(...)` を叩けば実戦の解決→開示儀式が本物の順で走る（検証済 2026-06-22）。1920×1080・確定フレームで滑らか・カーソル/録画枠が写らない。
- **狙うテイク**＝#5は **clean hit（勝ち札フレア）**／KOは決め手→YOU WINスラム／CROWDはMAX着火。強70%命中なので開示はミスらないテイクを選ぶ。
- **手収録は保険**（自動録画がコケた時のみ）。スマホ/画面録画は実時間で楽だが解像度・録画枠・狙った瞬間で劣る。
- JSCombat拡張（`addons/jscombat`・arm64 dylib）必須＝実機Macで録る（CIヘッドレスは get_texture()=null で不可）。

### 制作パイプライン（内製・再現可）
1. OP作画6枚を ffmpeg で Ken Burns 動画化（`-framerate 30`＋`zoompan d=1`＝1入力1出力でズーム累積。`d=フレーム数`は激増バグ）。
2. 実機ゲームプレイ＝Godot Movie Maker（`--write-movie`）＋ 2P注入ハーネス（`Roster.two_player=true`で両者の手を仕込む）で狙った技列を1試合で生成（§6）。RNGは `_rng=Math.random`＝非決定論なので強70%が外れたら録り直し。
3. 字幕＝ffmpegに drawtext 無し → **ImageMagickでヒラギノ角ゴW6＋ドロップシャドウの透過PNG化 → ffmpeg overlay**。ゲームUIと衝突する開示/実況は上部(H*0.085)、世界観/CTAは下部(H*0.75前後)。**フェードは各PNGをフルレングスにループ→`format=rgba,fade=t=in/out:alpha=1`でα制御（enable不要）**。
4. ffmpeg concat で連結。
