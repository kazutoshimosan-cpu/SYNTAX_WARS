## ① 選択画面・動機1行 ×6（確定）

各キャラ名の近くに出す1行（UIの1行枠に収まる長さ）。三人称・断言形で統一。
※現状の `catch`（機械的キャッチ「被弾で解析…」等）とは**別枠の新規ロア行**。

| キャラ | 動機1行 |
|---|---|
| NEXUS-7 | 自分が何者かも知らぬまま——勝つほどに、何かが"還ってくる" |
| SHADOW | "解放してはならない"と、ただ一人 知っている |
| TITAN | 命じられるまま戦う——だが、あれは"兵器"ではなかった |
| NOVA QUEEN | 支配は望まない。大国と"対等"に立ってみせる |
| LYRA-X | 封印は独占。解き放つことこそ、人類への贈り物だ |
| BLAZE | 理由なんてない。ただ、ドカンが見たい |

設計メモ：
- NEXUS＝「"還ってくる"」でARCHITECTへの回帰を匂わせる最安のEN伏線（本人は断片だと知らない）。
- SHADOW⇔LYRA＝「解放してはならない」vs「解き放つことこそ」＝言葉のレベルで対。選択画面で隣接配置すると対立軸が見える。

## ② 勝利台詞（確定・ランダムプール）

勝利画面で、各キャラの**プールからランダムに1つ**表示。下の全案が生きる。

### 通常5体

**SHADOW**
1. 俺が立っている限り、封印は揺るがない。
2. 本当のことを知ったまま勝つ——それが、俺の孤独だ。

**TITAN**
1. 俺は兵士だ——まだ、そう信じている。
2. 制した。だが、お前を"兵器"と呼ぶ言葉が、俺にはない。

**NOVA QUEEN**
1. 計算は正確だった。次は、同じテーブルに座る番だ。
2. 支配ではない——"対等"を、今日また一歩証明した。

**LYRA-X**
1. また一つ、壁が薄くなった。ARCHITECTは人類のものだ。
2. 封印は独占だ。それを守る者を、私は止め続ける。

**BLAZE**
1. 燃えてる時だけ、生きてる心地がすんだよなァ！　（ロア確定文）
2. なんで戦うかって？ドカンが見たいからに決まってんだろ！

### NEXUS-7（段階変化）

アーケード進行で**段階を決定**→その段の**2案からランダム**。単発VS／2P＝序盤プールを使う。
段階の目安（実装時に調整可）：序盤＝1〜2戦目／中盤＝3〜4戦目／終盤・FINAL＝5戦目（ラスボス）。

**序盤（無機質・任務報告）**
1. 脅威を排除。次の対象へ移行する。
2. 戦闘データ、記録完了。勝率は想定範囲内。

**中盤（違和感・ノイズ）**
1. 勝利を確認した。このデータノイズは——除去できない。何度試みても。
2. 排除完了。……なぜ、勝つほどに処理しきれない何かが積まれるのか。

**終盤／FINAL（断片が滲む）**
1. ……還る、のか。この感覚に——名前が、あった気がする。
2. これが最後ではない気がする。——初めての、確信だ。

設計メモ：
- NEXUS終盤「還る」は①動機「何かが"還ってくる"」＋canon「自分自身に還り着く」と一本の線。
- 直接「私はARCHITECTだ」とは言わない＝匂わせ止め。

## ③ CROWD ON FIRE 実況（確定・純ハイプ）

観客ゲージ最大時の着火で出す**実況/アナウンサーの煽り**。共通プールからランダム1つ。

★**実況は表（興行）側＝大衆向け放送。裏の懸け金（ARCHITECTアクセス権・代理戦争・AXIOM）は
大衆には伏せられている**ので実況にロアは混ぜない（ロアは勝利台詞＝当事者と、選択画面＝プレイヤー視点のみ）。
「世界中が放送で見ている」「見世物」レベルは大衆も知る"表の事実"なのでOK。

1. 場が沸いた——今夜の観衆は、世界そのものだ！
2. この熱量、限界を超えた——まだ終わるな！
3. 誰も座っていられない——立て、叫べ！
4. 世界中が、この一戦に釘付けだ！
5. 会場が揺れている——歓声が止まらない！
