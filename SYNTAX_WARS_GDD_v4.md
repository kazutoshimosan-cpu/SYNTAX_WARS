# SYNTAX WARS
## Game Design Document v4.0
_v3からの変更点：カウンター仕様刷新・通常行動のキャラ固有化_

---

## 1. ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | SF格闘ゲーム（ターン制コマンド選択） |
| プラットフォーム | ブラウザ（HTML / CSS / JavaScript） |
| プレイ人数 | 1人（vs CPU） / 2人（同端末 or ネット対戦：Firebase予定） |
| CPU難易度 | EASY / NORMAL / HARD |
| 操作方法 | ボタン選択のみ（テキスト入力なし） |
| 実装フェーズ | Phase 1〜5の段階実装（詳細は章8） |

### 基本ルール
- **行動選択：同時選択・同時解決**。両プレイヤーは同じターンに行動を同時選択し、選択確定後に両者の行動が同時に解決される。
- **勝利条件：** 相手のHPを0にした方が勝利。
- **引き分け：** 同ターンに両者HPが0になった場合は引き分け。リザルト画面に「DRAW」を表示し、リマッチ or タイトルへ戻るを選択できる。
- **試合形式：** 1本勝負。

## 2. 行動システム

### 2-1. ボタンレイアウト（4列）
※ スマホ対応のため各列は2×2グリッドで表示

| 列 | カテゴリ | 内容 |
|----|----------|------|
| 1列目 | 通常行動 | 弱・中・強・ガード（BLAZEのみ「起爆準備」）※技名はキャラごとに異なる |
| 2列目 | 属性付き攻撃 | 弱+速度特攻・中+ガード破壊・強+回復封じ・カウンター |
| 3列目 | 特殊行動 | 加速・スロー・回復・挑発・オーバークロック |
| 4列目 | 固有技 | キャラクター専用の必殺技（各1種） |

### 2-2. 通常行動（ゲージ不要）

技名はキャラごとに異なるが、ベースの数値は共通。キャラ補正がかかって最終ダメージが変わる。

| 行動 | ベースダメージ | ヒット率 | 備考 |
|------|---------------|----------|------|
| 弱 | 8 | 95% | 安定ヒット。キャラ補正後のダメージで計算 |
| 中 | 15 | 80% | 標準ダメージ。外れリスクあり |
| 強 | 22 | 78% | 高ダメージ・外れやすい。ジャストガードされるとゲージ+20%取られる |
| ガード | — | — | 次の被ダメージ50%軽減。BLAZEのみ「起爆準備」に置き換え |

**キャラ別の技名（例）**

| キャラクター | 弱 | 中 | 強 |
|--------------|----|----|-----|
| NEXUS-7 🤖 | ナノブレード | EMP波動 | 装甲展開 |
| SHADOW 🥷 | 影斬り | 毒針 | 煙幕 |
| NOVA QUEEN 🔮 | 次元裂き | 重力場 | 宇宙嵐 |
| TITAN 🦾 | 鉄拳 | 地震砕き | 装甲バリア |
| LYRA-X 🧬 | 酸液噴射 | 寄生体 | 変異爪 |
| BLAZE 🔥 | 火炎放射 | クラスター爆弾 | 溶岩弾 |

※ 技名はUI上のボタンラベルに使用。効果・ダメージはベース値にキャラ補正を適用して計算。

### 2-3. 属性付き攻撃（ゲージ25%消費 / 必中）

| 行動 | ダメージ | 属性効果 | 対策できる相手の戦略 |
|------|----------|----------|----------------------|
| 弱 + 速度特攻 | 8 + SPEEDボーナス | 相手のSPEEDが高いほど追加ダメージ増加（機雷的役割） | 加速ゴリ押し |
| 中 + ガード破壊 | 15 | ガードを無視してダメージ | ガード多用 |
| 強 + 回復封じ | 22 | 次ターン回復・挑発を封じる（1ターン） | 回復待ち |
| カウンター | — | 相手の攻撃を受けた際に同等ダメージを返す。自分のSPEEDが低いほど自分も被ダメあり | 攻撃力の高いキャラ全般 |

**カウンターの詳細仕様（v4変更）**
- **発動条件：** カウンター選択ターンに相手が攻撃してきた場合に発動（速度特攻に限らず全攻撃に反応）
- **返すダメージ：** 相手の攻撃ダメージと同等（等倍返し・ヒット率80%）
- **自分が受けるダメージ：** 相手の攻撃ダメージ × SPEED差に応じた割合

| 相手SPEED − 自分SPEED | 自分が受けるダメージ |
|----------------------|-------------------|
| 0以下（自分が同速or速い） | 0%（完全シャットアウト） |
| 1〜19 | 0% |
| 20〜39 | 10% |
| 40〜69 | 20% |
| 70以上 | 30% |

- **コンセプト：** 基本的には相手の攻撃を完全に防ぎつつ等倍返し。自分のSPEEDが相手より大幅に低い場合のみ、かすりダメージを受ける。BLAZEは攻撃+50%のためカウンターに特に弱い。

※ LYRA-Xの回復封じは2ターン持続（パッシブ特性）
※ NOVA QUEENの属性付き攻撃は次ターンのゲージコストが半減（パッシブ特性）

### 2-4. 特殊行動（ゲージ30%消費）

| 行動 | 効果 | 備考 |
|------|------|------|
| 加速 | 自分のSPEEDを+20〜30上昇（初期50のキャラが1回でFAST到達） | FAST状態になると速度特攻で追加ダメージを受けるリスクあり |
| スロー | 相手のSPEEDを大幅に低下 | LYRA-Xは効果が2ターン持続。SLOW状態は相手ダメージ-20% |
| 回復 | HPを15〜25回復 | 回復封じ中は使用不可 |
| 挑発 | 次の自分の攻撃ダメージ+30% | 回復封じ中は使用不可 |
| オーバークロック | ゲージ100%消費で次ターン強制2回行動 | 使用後3ターンはゲージ増加量が50%に減少 |

### 2-5. 固有技（ゲージ50%消費 / 必中）
※ v4変更：各キャラ1種に絞る。残り3つの技名は通常行動に割り当て済み。

| キャラクター | 固有技 | 効果 |
|--------------|--------|------|
| NEXUS-7 🤖 | 量子ビーム | 命中時ゲージ+20%・「量子強化」（次ターン属性攻撃コスト0%、1回限り）付与 |
| SHADOW 🥷 | 影分身 | 次の相手の攻撃を1回完全回避（確定） |
| NOVA QUEEN 🔮 | 星屑砲 | 相手ゲージ-20%・相手の属性攻撃を1ターン封じる |
| TITAN 🦾 | 超電磁砲 | 自分の減少HP比例でダメージ増加（瀕死で逆転の一撃） |
| LYRA-X 🧬 | 細胞再生 | 与ダメージ30%をHP吸収 |
| BLAZE 🔥 | 爆裂拳 | 35ダメージ（自爆リスクあり） |

### 2-6. ジャストガード
相手が「強攻撃」を選択したターンに自分が「ガード」を選択していた場合に発動。
- 被ダメージを0にする
- 自分のゲージを+20%する

※ BLAZEは「起爆準備」がガードの代わりのため、ジャストガードは発動しない
※ カウンター選択中はジャストガードと干渉しない（別の効果として独立）

## 3. キャラクター一覧と特性

### 3-1. 基本パラメータ

| キャラクター | タイプ | HP | 初期SPEED | 特徴 |
|--------------|--------|----|-----------|------|
| NEXUS-7 🤖 | バランス型 | 100 | 50 | どのシステムも標準的。初心者向け |
| SHADOW 🥷 | 速度型 | 100 | 65 | 初期SPEEDが高くヒット率+10%。通常ダメージ-10%（速度特攻のボーナスダメージには適用されない）。SPEED自然回復なし（65を維持） |
| NOVA QUEEN 🔮 | 属性型 | 90 | 50 | HPがやや低いがコンボ火力が高い。HP50%以下で被ダメ-15% |
| TITAN 🦾 | パワー型 | 100 | 30 | 初期SPEEDが低くヒット率-10%。通常ダメージ+30% |
| LYRA-X 🧬 | デバフ型 | 100 | 50 | デバフ効果が長い。ゲージ増加-20% |
| BLAZE 🔥 | 爆発型 | 100 | 50 | 全攻撃+50%だが自爆リスクあり。カウンターに弱い |

### 3-2. パッシブ特性（全キャラ）

**NEXUS-7 🤖 ／ アドバンスド・ラーニング**
- 相手が使った属性攻撃を学習し、次ターンに自分が同じ属性を使うと威力+50%（「学習済み」付与）
- ※ 量子ビーム命中時に付与される「量子強化」（コスト0%）とは別ステータス

---

**SHADOW 🥷 ／ 絶影（ぜつえい）**
- SPEED差が30以上ある時、10%の確率で相手の攻撃を完全回避する
- 通常ダメージ-10%（速度特攻のボーナスダメージには-10%軽減を適用しない＝ボーナスはフルで入る）
- 2回行動時の2回目の攻撃は必ずクリティカル（1.3倍）
- **SPEED自然回復なし：** 毎ターン終了時のSPEED50への自然回復が発動しない。初期SPEED65を維持し続ける。
- コンセプト：速さを軸にした読み合い特化キャラ。通常攻撃は弱いが速度特攻とカウンターで立ち回る。

---

**NOVA QUEEN 🔮 ／ コズミック・共鳴**
- ゲージが70%以上の時、属性付き攻撃のコストが25%→15%に半減
- HPが90とやや低いため、攻め続けることが前提のキャラ
- ※ 実装時注意：ゲージ70%の閾値を毎ターン判定し、ボタンのコスト表示をリアルタイムで更新すること

---

**TITAN 🦾 ／ 不動の要塞**
- SLOW状態（SPEED 35以下）の間、被ダメージを常時25%カットする
- SLOW時は攻撃-20%と被ダメ-25%が同時適用される（意図的トレードオフ）
- 相手はスローを使うとTITANを硬化させるジレンマに陥る
- カウンターと相性が良い：低SPEEDでもダメージ覚悟で大ダメージを返せる
- **オーバーヒート：** FAST状態（SPEED 70以上）になると被ダメ軽減が消え、通常攻撃が-10%になるが、ガード不可になる。「遅くて硬い」か「速くて軽いがガード貫通」のトレードオフ。

---

**LYRA-X 🧬 ／ バイオ・エボリューション**
- ターン終了時、固定3HPを自動回復する
- スロー・回復封じの効果が2ターン持続（他キャラは1ターン）
- SLOW状態の相手に毎ターン終了時に固定5ダメージ（腐食効果）
- 細胞再生：与えたダメージの30%をHP吸収

---

**BLAZE 🔥 ／ エクスプローシブ（爆弾魔）**
全攻撃ダメージ+50%。ただし攻撃時20%の確率で「自爆」が発生。

| 自爆の仕様 | 内容 |
|------------|------|
| 自爆発生確率 | 攻撃時20% |
| 自爆ダメージ | 与えたダメージの30%を自分も受ける |
| 誘爆 | 自爆発生時、自爆ダメージの50%を相手にも与える（道連れ効果） |
| 自爆のボーナス | 発生ターンのゲージ増加量が2倍 |
| 自爆の回避 | 起爆準備を使った次のターンは自爆率0%・確定クリティカル |

**BLAZE 🔥 ／ 起爆準備（ガードの置き換え）**
- そのターンは攻撃しない代わりに受けるダメージを30%軽減
- 次のターンの確定クリティカル（ダメージ1.3倍）＆自爆率0%
- **起爆準備の効果は「攻撃系行動（種類問わず）」をした瞬間に消費される。加速・スロー・回復・挑発などの非攻撃行動では持続する。**
- カウンターに弱い：攻撃+50%のため、相手にカウンターを構えられると大ダメージが返ってくる。起爆準備で様子見するのがカウンター対策。

## 4. 速度システム（SPEED）

### 4-1. 基本仕様

| 条件 | 効果 |
|------|------|
| SPEED差10以上 | 速い方が先に攻撃できる（先手表示あり） |
| SPEED差70以上 | 速い方が同ターン内に2回行動できる |
| SPEED 35以下（SLOW） | 自分の攻撃ダメージが-20%。TITANは被ダメージも-25% |
| SPEED 70以上（FAST） | 速度特攻の追加ダメージを受けるリスクあり |
| 毎ターン終了時 | SPEED50以上の場合のみ、50との差の15%分だけ50に向かって戻る。SPEED50以下は維持。 |

### 4-2. 速度特攻のボーナスダメージ
「弱+速度特攻」のボーナスダメージは相手のSPEEDに比例して増加する。

| 相手のSPEED | 追加ダメージ |
|-------------|-------------|
| 70〜74（FAST） | +8 |
| 75〜84 | +14 |
| 85〜100 | +20 |

### 4-3. オーバークロック
- コスト：ゲージ100%消費
- 効果：次のターンのみ2回行動が確定
- リスク：使用後3ターンはゲージ増加量が50%に減少

## 5. ゲージシステム

### 5-1. ゲージ初期値・増加ルール
**ゲーム開始時のゲージは30%**

| タイミング | 増加量 |
|------------|--------|
| 通常攻撃・属性攻撃・固有技を使用時 | +8% |
| ジャストガード成功時 | +20% |
| BLAZE自爆発生ターン | 通常増加量×2 |
| オーバークロック使用後3ターン | 増加量50%に減少 |
| HP40%以下の時（全キャラ共通） | ゲージ増加量×1.5倍（瀕死からの逆転を後押し） |

### 5-2. ゲージコスト一覧

| 行動カテゴリ | コスト |
|--------------|--------|
| 属性付き攻撃 | 25% |
| 加速 | 15% |
| 特殊行動（スロー・回復・挑発） | 30% |
| オーバークロック | 100% |
| 固有技 | 50% |

## 6. じゃんけん相性表

| 相手の戦略 | 有効な対抗手段 | 理由 |
|------------|----------------|------|
| 加速ゴリ押し | 弱 + 速度特攻 | FAST状態ほど追加ダメージが大きい |
| ガード多用 | 中 + ガード破壊 | ガードを無視して通常ダメージ |
| 回復待ち | 強 + 回復封じ | 次ターンの回復を封じる |
| 高火力攻撃（BLAZE等） | カウンター | 攻撃ダメージをそのまま返す。BLAZEには特に有効 |
| スロー連打 | 加速 → 攻撃 or オーバークロック | 速度を回復してからダメージを出す |
| オーバークロックを溜めてくる | 速度特攻 or ダメージを稼ぐ | 3ターンゲージ半減中は隙あり |
| BLAZEの起爆準備待ち | スロー or 回復封じ | 次ターンの確定クリティカルを崩す |
| カウンター構え | 起爆準備（BLAZE）or 攻撃しない択 | 攻撃しなければカウンターは発動しない |

## 7. ターン終了時の処理順序

同一ターン終了時に複数の効果が重なった場合、以下の順序で処理する：

1. **ダメージ系**（腐食など）
2. **回復系**（自動回復など）
3. **ステータス変化**（SPEEDの自然回復など）
4. **持続ターンのカウントダウン**（各ステータス異常の残りターン数を-1）

## 10. オーディエンスシステム

### 世界観
SYNTAX WARSは未来の地下闘技場で行われるバトル。観客の熱狂度がゲームに影響する。

### オーディエンスゲージ
読み勝ちが決まるたびにオーディエンスゲージが上昇する。

| トリガー | 上昇量 |
|----------|--------|
| ジャストガード成功 | +3 |
| カウンター成功 | +3 |
| 速度特攻ヒット（FASTボーナスあり） | +2 |
| ガード破壊ヒット | +2 |

ゲージが最大（10）に達すると「🔥 CROWD ON FIRE!」が発生してリセット。

### ランキングポイントボーナス（ネット対戦実装時）
CROWD ON FIRE状態で試合に勝利すると、通常より多くのランキングポイントを獲得できる。
