## 方針

- **トーン＝表の興行（薄い見せ方）**。「世界が見ている見世物トーナメント」の空気だけを出す。
  **裏（代理戦争・AXIOM支配・ARCHITECTの正体・NEXUSが覚醒した断片であること）は伏せる**＝エンディングで効かせる（OP=セットアップ／ED=ペイオフで twist 温存）。
- シネマティック横構図（16:9）・**6パネル**。クリック/Enterで順送り・**スキップ可**。
- 再生タイミング＝**初回起動で1回だけ自動再生**（`Progress` フラグ）＋**タイトルの「PROLOGUE」からいつでも再生**。
- セリフ／キャプションは**今は日本語のみ**。i18nキー化は出荷前ship prepで一括（[[project-i18n-localization-timing]]）。
- 各パネル記法：`P#〔ショット〕画の中身 ／ セリフ・キャプション ／ 色・トーン`。
- **ストアのタグライン＝「全部、同じ瞬間に開く。読み切った方が、勝つ。」を P4（伏せ）→P5（同時開示）の2枚に割る**＝"開く宣言"→"開く絵"で食い違いゼロ・コア機構を最強に見せる。OPがそのままトレーラーの頭になる。
- img パス＝`res://assets/art/opening/op_p<N>.png`（新規 `assets/art/opening/`）。

> **★2026-06-21 改訂（3部門レビュー反映）**：P4を2枚（伏せ→同時開示）に分割し、旧P5（NEXUS義眼CU）はカット（最高コスト×初見価値ゼロ＝EN伏線はED/タイトルidleに任せる）。P6サブワードを「FIGHT」→「**READY?**」（格ゲー誤認回避）。P2/P3キャプション推敲。6枚維持・純増ゼロ。

## OP「開幕」（6枚）

**P1**〔ワイド・やや俯瞰〕満員の巨大アリーナ。観客席の向こう、無数の中継スクリーンに同じ映像。世界中が、この一戦を見ている。
　キャプション：「——世界が、見ている。」
　色：ステージの二色照明（左マゼンタ／右シアン）＋客席の無数の光点。

**P2**〔インサート／放送テロップ〕中継画面のテロップ。賞品は"封じられた叡智"への鍵——という見世物のフレーム（※裏の意味は出さない）。
　キャプション：「勝者はひとり。手にするのは——封じられた叡智の、鍵。」
　色：放送グラフィックのクール系／白テロップ。

**P3**〔群像／ラインナップ〕6人の挑戦者が一画面に。表情は読めない。それぞれの思惑を、胸に。
　キャプション：「集うのは、6人。それぞれの思惑を、胸に抱えて。」
　色：各キャラ固有ネオン色が並ぶ＝多色。
　**並び順＝左から LYRA / BLAZE / SHADOW / NEXUS / NOVA / TITAN**（TITANは他の約2倍体格・右端）。

**P4**〔対面・伏せ札〕二者が至近で対峙。**互いの手はまだ伏せられている**＝開く直前の溜め。
　キャプション：「ここでは、全部が同じ瞬間に開く。」
　色：緊張の青白い"間"／ED寄りの抑えた暗さ・二人の輪郭だけ固有色。**対＝左BLAZE（暖）・右NEXUS（冷）**（タイトル画面と同じ対）。

**P5**〔同時開示・激突〕伏せられていた手が、**同時に開く**瞬間。両者の一手が一斉に晒され、ぶつかる。ゲームの肝＝同時開示。
　キャプション：「読み切った方が、勝つ。」
　色：開示の一閃／P4と同じ対のまま、シアン×暖色が交差。タグラインの payoff。

**P6**〔タイトルロゴ〕引き切り→"SYNTAX WARS" ロゴ。下に「**READY?**」（同時開示の掛け声＝「読めたか？」に直結／"FIGHT"は格ゲー誤認を呼ぶため不採用）。
　キャプション：（無／ロゴのみ）
　色：暗転からロゴが灯る。中央はロゴ配置用に暗く空ける。

> 着地＝「最強決定戦」ではなく「**読み合いの見世物**」としてフックする。P1 見世物感 → P4→P5 メカ提示（同時開示）→ P6 ロゴ、でトレーラー頭に直結。
> 流用方針＝**P3はテキストでシルエット指定**（既存idleはスプライトシート＝参照添付は逆効果）。新規作画は P1・P2・P3・P4・P5・P6（実質全枚）＝ただし軽量演出として枠でも成立。
