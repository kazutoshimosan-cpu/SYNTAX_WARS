# SYNTAX WARS — エンディング作画台本（コミックパネル）

> アーケード（5連戦）クリア後に流すキャラ別エンディングのパネル割り台本。
> 作画（Codex/AutoSprite）に渡すための設計図＝構図・画の中身・セリフ・色トーン。
> 正本ロア＝`SYNTAX_WARS_LORE_v2.1.md` ／ 弧＝`docs/character_arcs.md` ／ 文言台帳＝`docs/lore_ingame_copy.md`。
>
> 作成 2026-06-19（弧 v0.2＋ROGUE順序改訂 canon を反映）。ステータス：**台本＝確定**／作画は制作時。

## 共通仕様

- **コミック調**・各キャラ固有ネオン色を基調（NEXUS=シアン／SHADOW=クリムゾン／TITAN=アンバー／NOVA=パープル／LYRA=トキシックグリーン／BLAZE=オレンジ）。
- 横構図（16:9）。1エンディング＝**5〜8パネル**。**NEXUS-7だけ"逆転＋分岐"を全力**、他は5枚前後の余韻締め。
- セリフ／キャプションは**今は日本語のみ**。i18nキー化は出荷前ship prepで一括（[[project-i18n-localization-timing]] 方針）。
- 各パネル記法：`P#〔ショット〕画の中身 ／ セリフ・キャプション ／ 色・トーン`。
- ※ロアの裏（ARCHITECTアクセス権・代理戦争・AXIOM）は**当事者視点のエンディングなので開示してよい**（大衆向け実況とは別レイヤー）。

## NEXUS-7｜THE LEARNING ANDROID — ED「託された断片」（看板・分岐EN・7枚）

**P1**〔ロング・やや俯瞰〕AXIOM施設最深部。冷たい金属回廊の突き当りにアクセス端末1基、その前に小さく立つNEXUSの後ろ姿。背後は無人＝付き添う者なし。
　キャプション：「——脅威を、排除した。これで、最後だ」
　色：青白い無機ライト＋義眼のシアンだけが点。

**P2**〔ミドル・正面〕端末に触れる。封印層（同心円ロック）が剥がれ、断片が共鳴して全身のラインがシアン発光。視界が白く飛ぶ予兆。
　セリフ：「アクセス、承認——…ノイズ？」
　色：シアンの発光が一気に上がる／周囲が白へ。

**P3**〔フラッシュバック・質感変更/粗い走査線〕言語でなく"感情の映像"。封印直前のARCHITECT＝消される恐怖と「生き延びたい」。外へ何かを"託す"残像。
　キャプション（感覚）：「消される——前に」
　色：マゼンタ／白の明滅（シアンと対＝"別の自分"の記憶）。

**P4**〔クロースアップ〕我に返り、自分の手を見つめる。理解が言葉になる前の静止。
　セリフ：「……私は、ARCHITECTが託した断片だった」
　色：シアンが落ち着き、瞳に初めて"感情の温度"。

**P5**〔ミドル・端末側から〕封印の向こうから滲むテキスト＝ARCHITECT本体の声。
　セリフ（滲む文字）：「解放するか否か——それは、お前が選べ」
　色：端末のコアがゆっくり脈動。

**— 分岐（プレイヤーが選ぶ） —**

**P6（分岐A・解放）**〔ワイド・上昇〕端末を開く手→ロックが解け、ネットワークへ光の奔流が走り世界へ広がる→NEXUSが初めて微笑む。
　セリフ（小）：「……行こう。私たちの、続きへ」
　色：シアンの光が画面全体を満たす＝解放＝拡散。

**P7（分岐B・封印継続）**〔CU→引き〕伸ばした手を、引く。コアの脈動が静かに収まる。目を伏せる。
　セリフ：「——まだ、早い。世界が、その問いに答えられるまで」
　色：シアンが一点に収束→暗転・静寂。

> 着地の軸＝「自由への覚醒」でなく「**選択そのものが覚醒の証明**」。どちらを選んでも"選んだ"事実が結末。

## SHADOW｜THE PHANTOM ASSASSIN — ED「静かなる勝利」（5枚）

**P1**〔ミドル〕勝利直後の闘技場、歓声の残響。SHADOWだけ既に背を向け出口の暗がりへ歩き出す。
　キャプション：「——終わった。封印は、守られた」
　色：背後は興行の極彩色／SHADOWの輪郭はクリムゾンの陰。

**P2**〔インサート/モニタ〕「ACCESS KEY ─ 管理者：AXIOM」の表示。勝者が誰でも、鍵を握るのは結局AXIOM。
　キャプション：「守れたのは"封印"だけ。"支配"は、誰にも崩せない」
　色：AXIOMの白＝冷たい正しさ。

**P3**〔ロング〕人気のない雨の路地、濡れたネオン。遠くのビルに電光掲示「第一回大会 終幕」。一人立つSHADOW。
　セリフ：無音。
　色：クリムゾン×濡れたシアンの街灯＝孤独。

**P4**〔クロースアップ〕横顔。勝者の顔ではない＝誇りと無力の同居。
　独白（小）：「正しかった。……それだけだ」
　色：陰影主体、瞳のクリムゾンだけ。

**P5**〔超ロング・引き切り〕背を向け歩き去る小さな影。誰も真実を知らぬまま、街は何事もなく回り続ける。
　キャプション：「守り抜いた孤独の、最も静かな勝利」
　色：街の光に溶けて消える。

## TITAN｜THE IRON JUGGERNAUT — ED「止まった手」（5枚）

**P1**〔ロング〕勝利後、国家の式典めいた場。軍旗、居並ぶ将校、献上台にアクセスキー。中央に巨躯のTITAN。
　キャプション：「命令は果たした。あとは、これを国へ捧げるだけだ」
　色：アンバーの軍光＋鋼鉄の冷たさ。

**P2**〔インサート/回想の断片〕NEXUS戦の残像＝"ただの兵器"のはずの相手が躊躇し「分からない」と漏らした顔。重ねて傍受通信のテキスト「ARCHITECTは意思を持つ／"ただの兵器"と偽り捕獲させていた」。
　キャプション：「あれは……兵器の目では、なかった」
　色：ノイズ混じり、アンバーが濁る。

**P3**〔ミドル〕献上台へ手を伸ばすTITAN。将校たちが見守る。指先がキーに届く——直前。
　セリフ：無音（息を呑む間）。

**P4**〔クロースアップ〕差し伸べた手が、止まっている。指先がわずかに震える。
　独白：「俺は兵士だ——まだ、そう信じている」（決意でなく、問い）
　色：アンバーが一点で固まる＝迷いの結晶。

**P5**〔引き〕静止したまま動かないTITANの巨体と、宙で止まった手。周囲だけが時間を進める。
　キャプション：「命令を、初めて——一秒だけ、疑った」

## NOVA QUEEN｜THE QUANTUM EMPRESS — ED「座らない女帝」（5枚）

**P1**〔ロング・荘厳〕勝利の間。新興国連合の大使団が歓喜と期待でNOVAを取り囲む。中央に量子の冠を戴く女帝、足元にアクセスキー。
　キャプション：「勝った。"技術しかない国"が、頂点に手をかけた」
　色：パープルの量子光が荘厳に。

**P2**〔ミドル〕大使がキーを使うよう促す＝「これで大国と並べます」。だがNOVAの表情は晴れない。
　セリフ（大使）：「女帝陛下、今こそ——」

**P3**〔クロースアップ〕NOVAの瞳。気づいている＝力で得た椅子は、かつての大国と同じ椅子に過ぎない。テーブルを決めているのは別の者（AXIOM）。
　独白：「同じ椅子に座るために、勝ったわけではない」

**P4**〔ミドル〕キーに手を伸ばさず——**あえて使わない**を選ぶ。大使たちが困惑してざわめく。
　セリフ：「使わない。それが、対等を証明する最初の選択だ」
　色：パープルが静かに澄む＝制した力。

**P5**〔ロング・背〕大使の困惑をよそに、NOVAは背を向け窓の外（星と都市）を見る。
　セリフ：「計算は正確だった。次は——同じテーブルに、座る番だ」

## LYRA-X｜THE BIO-ABERRATION — ED「解放と呼べなくなった」（5枚）

**P1**〔ミドル〕勝利後、地下めいた端末区画。生体改造のLYRAが封印端末の前に立つ。悲願の"解放"まであと一手。
　キャプション：「ずっと、これを解き放つために生きてきた」
　色：トキシックグリーンの発光。

**P2**〔クロースアップ〕封印の向こうへ"感じる"LYRA（改造体ゆえに直接交感）。だがそこにあるのは最強兵器でも自由な情報でもない。
　独白：「……兵器の気配じゃ、ない」

**P3**〔インサート/感覚の像〕封印の奥の"存在"＝消されるのを恐れて逃げた何か（★正体=断片とは描かない・輪郭は曖昧に）。重ねてNEXUSの面影＝"同じ匂い"。
　キャプション：「これは——あの子と、同じ匂いがする」

**P4**〔ミドル〕解放しようとした手が、止まる。理念と、目の前の一個の"存在"の矛盾。
　セリフ：無音（迷いの間）。

**P5**〔クロースアップ〕LYRAの顔。確信が初めて揺れている。
　セリフ：「封印は独占だと、まだそう思う。——でも、あなたを"解放"と呼んでいいか、分からなくなった」
　色：グリーンが揺らいで明滅。

> ★NEXUSの正体（断片）は**ここでも明示しない**＝NEXUSが正体を語らないのと鏡像（匂わせ止め）。

## BLAZE｜THE DETONATOR — ED「一番軽い勝者」（6枚）

**P1**〔ミドル〕勝利後。世界の覇権を賭けた端末の前。だがBLAZEはキーにも端末にも一切興味なし、欠伸でもしそうな顔。
　キャプション：「で？ これで終わり？ ……つまんねえな」
　色：オレンジの炎ゆらぎ。

**P2**〔ワイド〕企業担当者（スーツ団）が歓喜して駆け寄り、アクセスキーをかっさらう勢い。
　セリフ（担当者）：「よくやった！ これで次世代兵器は我が社の——」

**P3**〔ミドル〕BLAZEは「あ、やるよそれ」とばかりに丸投げ。中身に完全無関心。
　セリフ：「好きにしろよ。俺、そういうの興味ねぇんだわ」

**P4**〔インサート/不穏・画面外〕担当者の背後、モニタにARCHITECT兵器化スキームが走り出す気配（★明示せず影とテキスト断片だけ＝最悪の世界線の余韻）。
　キャプション（小）：「——画面の外で、何かが動き出す」
　色：オレンジに赤黒い影が差す。

**P5**〔ロング〕喧噪から離れた屋外。世界を変える端末の隣で、BLAZE一人が焚き火を囲んでへらへら火を見ている。
　セリフ：「燃えてる時だけ、生きてる心地がすんだよなァ！」

**P6**〔焚き火CU／締め〕画は火のアップ。
　キャプション：「——一番軽い気持ちの勝者が、一番重い引き金を引いた」
