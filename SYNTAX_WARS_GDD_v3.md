# SYNTAX WARS
## Game Design Document v3.0
_v2からの変更点：デバッグレビュー反映・仕様確定版_

---

## 1. ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | SF格闘ゲーム（ターン制コマンド選択） |
| プラットフォーム | ブラウザ（HTML / CSS / JavaScript） |
| プレイ人数 | 1人（vs CPU） / 2人（同端末 or ネット対戦：Firebase予定） |
| CPU難易度 | EASY / NORMAL / HARD |
| 操作方法 | ボタン選択のみ（テキスト入力なし） |
| 実装フェーズ | Phase 1〜5の段階実装（詳細は章8） |

### 基本ルール（v3追加）
- **行動選択：同時選択・同時解決**。両プレイヤーは同じターンに行動を同時選択し、選択確定後に両者の行動が同時に解決される。
- **勝利条件：** 相手のHPを0にした方が勝利。
- **引き分け：** 同ターンに両者HPが0になった場合は引き分け。リザルト画面に「DRAW」を表示し、リマッチ or タイトルへ戻るを選択できる。
- **試合形式：** 1本勝負。

## 2. 行動システム

### 2-1. ボタンレイアウト（4列）
※ スマホ対応のため各列は2×2グリッドで表示

| 列 | カテゴリ | 内容 |
|----|----------|------|
| 1列目 | 通常行動 | 弱・中・強・ガード（BLAZEのみ「起爆準備」） |
| 2列目 | 属性付き攻撃 | 弱+速度特攻・中+ガード破壊・強+回復封じ・**カウンター** |
| 3列目 | 特殊行動 | 加速・スロー・回復・挑発・オーバークロック |
| 4列目 | 固有技 | キャラクター専用の必殺技（各4種） |

### 2-2. 通常行動（ゲージ不要）

| 行動 | ダメージ | ヒット率 | 備考 |
|------|----------|----------|------|
| 弱攻撃 | 8 | 95% | 安定ヒット。属性なしでも手堅くダメージを稼げる |
| 中攻撃 | 15 | 80% | 標準ダメージ。外れリスクあり |
| 強攻撃 | 22 | 65% | 高ダメージ・外れやすい。ジャストガードされるとゲージ+20%取られる |
| ガード | — | — | 次の被ダメージ50%軽減。BLAZEのみ「起爆準備」に置き換え |

### 2-3. 属性付き攻撃（ゲージ25%消費 / 必中）
※ v2変更：コストを30%→25%に引き下げ

| 行動 | ダメージ | 属性効果 | 対策できる相手の戦略 |
|------|----------|----------|----------------------|
| 弱 + 速度特攻 | 8 + SPEED差ボーナス | 相手のSPEEDが高いほど追加ダメージ増加 | 加速ゴリ押し |
| 中 + ガード破壊 | 15 | ガードを無視してダメージ | ガード多用 |
| 強 + 回復封じ | 22 | 次ターン回復・挑発を封じる（1ターン） | 回復待ち |
| **カウンター**（v3変更） | — | 速度特攻を受けたら自分のSPEEDが大幅上昇 | 速度特攻戦略 |

※ v3変更：「ガード+カウンター」→「カウンター」に改名。ガード効果を削除。カウンターはゲージ25%消費の純粋な読み行動として機能する。
※ LYRA-Xの回復封じは2ターン持続（パッシブ特性）
※ NOVA QUEENの属性付き攻撃は次ターンのゲージコストが半減（パッシブ特性）

### 2-4. 特殊行動（ゲージ30%消費）

| 行動 | 効果 | 備考 |
|------|------|------|
| 加速 | 自分のSPEEDを大幅に上昇 | FAST状態になると相手から「弱+速度特攻」で追加ダメージを受けるリスクあり |
| スロー | 相手のSPEEDを大幅に低下 | LYRA-Xは効果が2ターン持続。SLOW状態は相手ダメージ-20% |
| 回復 | HPを15〜25回復 | 回復封じ中は使用不可 |
| 挑発 | 次の自分の攻撃ダメージ+30% | 回復封じ中は使用不可 |
| オーバークロック | ゲージ100%消費で次ターン強制2回行動 | 使用後3ターンはゲージ増加量が50%に減少 |

### 2-5. 固有技（ゲージ50%消費 / 必中）
※ v2変更：コストを30%→50%に引き上げ・威力を28→35に強化

| キャラクター | 固有技一覧（各4種） | 特記事項 |
|--------------|---------------------|----------|
| NEXUS-7 🤖 | 量子ビーム / ナノブレード / EMP波動 / 装甲展開 | 量子ビーム：命中時ゲージ+20%・「量子強化」（次ターン属性攻撃コスト0%、1回限り）付与 |
| SHADOW 🥷 | 影斬り / 毒針 / 煙幕 / 影分身 | 影分身：次の相手の攻撃を1回完全回避（確定） |
| NOVA QUEEN 🔮 | 星屑砲 / 次元裂き / 重力場 / 宇宙嵐 | 星屑砲：相手ゲージ-20%・相手の属性攻撃を1ターン封じる |
| TITAN 🦾 | 鉄拳 / 地震砕き / 装甲バリア / 超電磁砲 | 超電磁砲：自分の減少HP比例でダメージ増加 |
| LYRA-X 🧬 | 酸液噴射 / 細胞再生 / 寄生体 / 変異爪 | 細胞再生：与ダメージ30%をHP吸収 |
| BLAZE 🔥 | 火炎放射 / クラスター爆弾 / 溶岩弾 / 爆裂拳 | 爆裂拳：35ダメージ（自爆リスクあり） |

### 2-6. ジャストガード
相手が「強攻撃」を選択したターンに自分が「ガード」を選択していた場合に発動。
- 被ダメージを0にする
- 自分のゲージを+20%する
- 読み勝った時のリターンを明確化し、守る楽しみを提供する

※ BLAZEは「起爆準備」がガードの代わりのため、ジャストガードは発動しない
※ v3変更：「カウンター」選択中は別の効果（速度特攻への反応）のためジャストガードとは干渉しない

## 3. キャラクター一覧と特性

### 3-1. 基本パラメータ

| キャラクター | タイプ | HP | 初期SPEED | 特徴 |
|--------------|--------|----|-----------|------|
| NEXUS-7 🤖 | バランス型 | 100 | 50 | どのシステムも標準的。初心者向け |
| SHADOW 🥷 | 速度型 | 100 | 65 | 初期SPEEDが高くヒット率+10%。ただし通常ダメージ-20%（速度特攻のボーナスダメージには適用されない） |
| NOVA QUEEN 🔮 | 属性型 | 90 | 50 | HPがやや低いがコンボ火力が高い。HP50%以下で被ダメ-15% |
| TITAN 🦾 | パワー型 | 100 | 30 | 初期SPEEDが低くヒット率-10%。通常ダメージ+30% |
| LYRA-X 🧬 | デバフ型 | 100 | 50 | デバフ効果が長い。ゲージ増加-20% |
| BLAZE 🔥 | 爆発型 | 100 | 50 | 全攻撃+50%だが自爆リスクあり |

### 3-2. パッシブ特性（全キャラ）

**NEXUS-7 🤖 ／ アドバンスド・ラーニング**
相手が使った属性攻撃を学習し、次ターンに自分が同じ属性を使うと威力+50%。「学習済み」状態が付与される。
- 例：相手が「中+ガード破壊」を使った次のターン、自分の「中+ガード破壊」が+50%になる
- ※ 量子ビーム命中時に付与される「量子強化」（コスト0%）とは別ステータス

---

**SHADOW 🥷 ／ 絶影（ぜつえい）**
- SPEED差が30以上ある時、10%の確率で相手の攻撃を完全回避する
- 通常ダメージ-20%（速度特攻のボーナスダメージには-20%軽減は適用されない＝ボーナスはフルで入る）
- 2回行動時の2回目の攻撃は必ずクリティカル（1.3倍）
- コンセプト：速さを軸にした読み合い特化キャラ。相手が速さに干渉しようとする択に対してことごとく有利を取れる

---

**NOVA QUEEN 🔮 ／ コズミック・共鳴**
- ゲージが70%以上の時、属性付き攻撃のコストが25%→15%に半減
- 属性攻撃 → コストの安くなった固有技というコンボルートが確立
- HPが90とやや低いため、攻め続けることが前提のキャラ
- ※ 実装時注意：ゲージ70%の閾値を毎ターン判定し、ボタンのコスト表示をリアルタイムで更新すること

---

**TITAN 🦾 ／ 不動の要塞**
- SLOW状態（SPEED 35以下）の間、被ダメージを常時25%カットする
- SLOW時は攻撃-20%と被ダメ-25%が同時適用される（意図的トレードオフ）
- 相手はスローを使うとTITANを硬化させるジレンマに陥る
- 超電磁砲：自分の減少HPに比例してダメージが増加（逆転の一撃）

---

**LYRA-X 🧬 ／ バイオ・エボリューション**
- ターン終了時、固定3HPを自動回復する
- スロー・回復封じの効果が2ターン持続（他キャラは1ターン）
- SLOW状態の相手に毎ターン終了時に固定5ダメージ（腐食効果）
- 細胞再生：与えたダメージの30%をHP吸収

---

**BLAZE 🔥 ／ エクスプローシブ（爆弾魔）**
全攻撃ダメージ+50%。ただし攻撃時20%の確率で「自爆」が発生。

| 自爆の仕様 | 内容 |
|------------|------|
| 自爆発生確率 | 攻撃時20% |
| 自爆ダメージ | 与えたダメージの30%を自分も受ける |
| 自爆のボーナス | 発生ターンのゲージ増加量が2倍 |
| 自爆の回避 | 起爆準備を使った次のターンは自爆率0%・確定クリティカル |

**BLAZE 🔥 ／ 起爆準備（ガードの置き換え）**
BLAZEのガードボタンは「起爆準備」に置き換えられる。
- そのターンは攻撃しない代わりに受けるダメージを30%軽減
- 次のターンの攻撃が確定クリティカル（ダメージ1.3倍）
- 次のターンの自爆率が0%になる
- **v3変更：起爆準備の効果は「攻撃系行動（種類問わず）」をした瞬間に消費される。加速・スロー・回復・挑発などの非攻撃行動では持続する。**

※ 通常のガード（50%軽減）より防御力は低いが、次の攻撃が大きく強化される

## 4. 速度システム（SPEED）

### 4-1. 基本仕様

| 条件 | 効果 |
|------|------|
| SPEED差10以上 | 速い方が先に攻撃できる（先手表示あり） |
| SPEED差70以上 | 速い方が同ターン内に2回行動できる |
| SPEED 35以下（SLOW） | 自分の攻撃ダメージが-20%。TITANは被ダメージも-25% |
| SPEED 65以上（FAST） | 速度特攻の追加ダメージを受けるリスクあり |
| 毎ターン終了時 | **SPEED50以上の場合のみ**、50との差の15%分だけ50に向かって戻る |

※ v2変更：2回行動の条件をSPEED差50→70に引き上げ（加速ゲー化を防止）
※ **v3変更：SPEEDの自然回復はSPEED50以上の時のみ発動。SPEED50以下のキャラ（TITANなど）は低いSPEEDがそのまま維持される。**

### 4-2. 速度特攻のボーナスダメージ
「弱+速度特攻」のボーナスダメージは相手のSPEEDに比例して増加する。

| 相手のSPEED | 追加ダメージ |
|-------------|-------------|
| 65〜74（FAST） | +8 |
| 75〜84 | +14 |
| 85〜100 | +20 |

### 4-3. オーバークロック
- コスト：ゲージ100%消費
- 効果：次のターンのみ2回行動が確定
- リスク：使用後3ターンはゲージ増加量が50%に減少
- ※ 鈍足キャラ（TITAN等）がここぞという場面で使う逆転手段

## 5. ゲージシステム

### 5-1. ゲージ初期値・増加ルール
※ **v3追加：ゲーム開始時のゲージは30%**

| タイミング | 増加量 |
|------------|--------|
| 通常攻撃・属性攻撃・固有技を使用時 | +8% |
| ジャストガード成功時 | +20% |
| BLAZE自爆発生ターン | 通常増加量×2 |
| オーバークロック使用後3ターン | 増加量50%に減少 |

### 5-2. ゲージコスト一覧

| 行動カテゴリ | コスト | 備考 |
|--------------|--------|------|
| 属性付き攻撃 | 25% | v2変更：30%→25%に引き下げ |
| 特殊行動（加速・スロー・回復・挑発） | 30% | 据え置き |
| オーバークロック | 100% | 新規追加 |
| 固有技 | 50% | v2変更：30%→50%に引き上げ・威力35に強化 |

## 6. じゃんけん相性表

| 相手の戦略 | 有効な対抗手段 | 理由 |
|------------|----------------|------|
| 加速ゴリ押し | 弱 + 速度特攻 | FAST状態ほど追加ダメージが大きい |
| ガード多用 | 中 + ガード破壊 | ガードを無視して通常ダメージ |
| 回復待ち | 強 + 回復封じ | 次ターンの回復を封じる |
| 速度特攻を狙ってくる | カウンター | 速度特攻を受けたら逆に自分が加速 |
| スロー連打 | 加速 → 攻撃 or オーバークロック | 速度を回復してからダメージを出す |
| オーバークロックを溜めてくる | 速度特攻 or ダメージを稼ぐ | 3ターンゲージ半減中は隙あり |
| BLAZEの起爆準備待ち | スロー or 回復封じ | 次ターンの確定クリティカルを崩す |

## 7. ターン終了時の処理順序（v3追加）

同一ターン終了時に複数の効果が重なった場合、以下の順序で処理する：

1. **ダメージ系**（腐食など）
2. **回復系**（自動回復など）
3. **ステータス変化**（SPEEDの自然回復など）
4. **持続ターンのカウントダウン**（各ステータス異常の残りターン数を-1）

## 8. 実装フェーズ

| フェーズ | 内容 | 優先度 |
|----------|------|--------|
| Phase 1（コア） | 基本バトル・4列ボタンUI・通常行動・ゲージ・速度システム・CPU AI | 高 |
| Phase 2（キャラ個性） | パッシブ特性・BLAZE自爆・起爆準備・固有技強化・ジャストガード | 高 |
| Phase 3（高度な読み合い） | オーバークロック・速度特攻ボーナス計算・NOVA QUEENコンボ | 中 |
| Phase 4（演出・拡張） | BGM・アニメーション強化 | 中 |
| Phase 5（ネット対戦） | Firebase Realtime Database・ルーム作成・入室システム | 低 |

## 9. ステータス異常一覧

| 異常名 | 付与条件 | 効果 | 持続 |
|--------|----------|------|------|
| FAST | 加速・固有技による速度上昇 | 先手・速度特攻の追加ダメージリスク | SPEED 65以上の間 |
| SLOW | スロー・速度特攻ヒット | ダメージ-20%。後手になりやすい | SPEED 35以下の間（LYRA-X付与は+1T） |
| 回復封じ | 強+回復封じがヒット | 回復・挑発が使えない | 1ターン（LYRA-X付与は2ターン） |
| カウンター待機 | カウンター選択（v3：ガード効果なし） | 速度特攻を受けたら自分のSPEEDが大幅上昇 | 次に速度特攻を受けるまで（毎ターン選択制） |
| 挑発 | 挑発選択ターン | 次の攻撃ダメージ+30% | 次の攻撃1回 |
| 自爆 | BLAZE攻撃時20%で発生 | 与ダメ30%を自分も受ける。ゲージ増加2倍 | その攻撃のみ |
| 起爆準備 | BLAZE「起爆準備」選択 | 確定クリティカル＆自爆率0% | **攻撃系行動（種類問わず）を使うまで持続（v3変更）** |
| 学習済み | NEXUS-7パッシブ発動 | 次ターン同属性攻撃の威力+50% | 次の同属性攻撃1回 |
| 量子強化 | NEXUS-7量子ビーム命中（v3：学習済みと分離） | 次ターン属性攻撃コスト0% | 次の属性攻撃1回 |
| 腐食 | LYRA-Xパッシブ（SLOW相手） | ターン終了時に5固定ダメージ | SLOW状態の間 |
| HP閾値パッシブ | NOVA QUEEN固有：HP50%以下 | 被ダメージ-15% | HP50%以下の間（常時） |
